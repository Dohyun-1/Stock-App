# StockPro 실행 방법 (MacBook)

## 실패했을 때 대처법

### 1. "EADDRINUSE" 또는 "address already in use" 에러

**원인**: 포트(3005 또는 4000)가 이미 사용 중입니다.
- 이미 서버가 실행 중일 수 있어요. **브라우저에서 http://localhost:3005 열어보세요.** 작동하면 그대로 사용하면 됩니다!
- 새로 시작하려면:

```bash
# 4000 포트로 실행 (다른 포트)
npm start
```
→ http://localhost:4000 접속

또는 3005 포트 사용 중인 프로그램 종료 후:
```bash
npm run start:3005
```

---

### 2. "command not found: npm" 에러

**원인**: Node.js가 설치되지 않았거나 터미널에서 찾지 못합니다.

**해결**: [nodejs.org](https://nodejs.org) 접속 → LTS 버전 다운로드 → 설치 → 터미널 **새로 열기**

---

### 3. "npm run build" 실패 (Failed to compile)

**해결**:
```bash
cd "/Users/dohyun/Desktop/무제 폴더/stock-hackathon"
rm -rf node_modules .next

npm run build
```
